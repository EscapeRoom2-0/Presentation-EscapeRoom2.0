========================================English===============================================
Hello Programmer, this is the Presentation-EscapeRoom 2.0 Project for ETEC Pedro Ferreira Alves.

Unlike the Escape Room game, this one has the standard source code

To make this program functional, follow these steps:

1. Run composer install to install all dependencies.

2. Copy the environment example file by running:

cp .env.example .env

3. Finally, apply the database migrations and seeders with:

php artisan migrate --seed

Now, you can enjoy the website/code! :)
